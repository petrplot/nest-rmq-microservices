### nest-rmq-microservices
solving a test task on the topic of creating a microservice on rmq

## Installation

```bash
# for http-getaway
$ cd http-getaway
$ npm install

# for rmq-service
$ cd rmq-service
$ npm install
```
## Running the app

```bash
# for http-getaway
$ cd http-getaway

# for rmq-service
$ cd rmq-service

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```
