# nest-rmq-microservices
solving a test task on the topic of creating a microservice on rmq. 
For the application to work correctly, you need to install the RabbitMQ message broker on your computer

## Installation

### for http-getaway
```
$ cd http-getaway
```
```
$ npm install
```
### for rmq-service
```
$ cd rmq-service
```
```
$ npm install
```
## Running the app


### for http-getaway
```
$ cd http-getaway
```

### for rmq-service
```
$ cd rmq-service
```
### development
```
$ npm run start
```
### watch mode
```
$ npm run start:dev
```
### production mode
```
$ npm run start:prod

```
